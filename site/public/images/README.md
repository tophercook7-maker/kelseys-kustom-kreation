# Image Assets

## Required Images

### Hero Background
- **File**: `hero-bg.jpg`
- **Size**: 1920x1080px or larger
- **Format**: JPG
- **Usage**: Full-canvas background for all pages

### Product Images
Place all product images in the `products/` directory:

#### Shirts
- `shirt-1.jpg` - Main product image
- `shirt-2.jpg` - Alternative view

#### Tumblers
- `tumbler-1.jpg` - Main product image
- `tumbler-2.jpg` - Alternative view

#### License Plates
- `plate-1.jpg` - Main product image
- `plate-2.jpg` - Alternative view

### Placeholder
- `placeholder.jpg` - Fallback image for missing products

## Image Guidelines

- Use high-quality, professional photography
- Neutral backgrounds preferred
- Consistent lighting across all images
- Premium, clean aesthetic
- Optimize file sizes for web (aim for <500KB per image)

## Adding Images

1. Add your hero background image as `hero-bg.jpg`
2. Add product images to the `products/` folder
3. Update image paths in `lib/products.ts` if filenames differ
4. Test images display correctly on all pages
