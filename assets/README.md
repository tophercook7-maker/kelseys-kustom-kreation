# Brand Assets

## Logo Files

### SVG Logo
- `logo.svg` - Scalable vector logo for web use

### PNG Logos
You'll need to generate PNG versions from the SVG:

**Light Version (logo-light.png)**
- White/light background
- Use for dark backgrounds
- Recommended size: 1200x240px (5:1 ratio)

**Dark Version (logo-dark.png)**
- Dark background
- Use for light backgrounds
- Recommended size: 1200x240px (5:1 ratio)

### Generating PNGs
You can use tools like:
- Inkscape (free): `inkscape logo.svg --export-filename=logo-light.png --export-width=1200`
- Adobe Illustrator
- Online SVG to PNG converters

## Product Images

Place product images in `site/public/images/products/`:

### Required Images:
- `shirt-1.jpg` - T-shirt front view
- `shirt-2.jpg` - T-shirt folded/alternative view
- `tumbler-1.jpg` - Tumbler main view
- `tumbler-2.jpg` - Tumbler alternative view
- `plate-1.jpg` - License plate flat lay
- `plate-2.jpg` - License plate mounted view

### Image Specifications:
- Format: JPG or PNG
- Recommended size: 1200x1200px (square) or 1200x1600px (portrait)
- Quality: High resolution, professional lighting
- Style: Clean, neutral backgrounds, premium feel

## Hero Background Image

Place your hero background image at:
- `site/public/images/hero-bg.jpg`

### Specifications:
- Format: JPG
- Recommended size: 1920x1080px or larger
- Aspect ratio: 16:9 or wider
- Quality: High resolution
- Should work as a full-canvas background
