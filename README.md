# Kelsey's Kustom Kreation

Premium e-commerce website for custom products.

## Project Structure

```
Kelseys-Kustom-Kreation/
├── assets/                 # Brand assets
│   ├── logo.svg           # Logo (SVG)
│   ├── logo-light.png     # Logo light version
│   ├── logo-dark.png      # Logo dark version
│   └── product-images/    # Product image placeholders
├── branding/              # Brand guidelines and documentation
│   ├── brand-guidelines.md
│   └── pricing.md
├── business-card/         # Business card design
│   └── business-card.html
└── site/                  # Next.js website
    ├── app/               # Application pages
    ├── components/        # React components
    ├── lib/               # Utilities
    ├── public/            # Static assets
    └── netlify.toml       # Netlify configuration
```

## Quick Start

1. Navigate to the site directory:
```bash
cd site
```

2. Install dependencies:
```bash
npm install
```

3. Set up environment variables (see `site/README.md`)

4. Add your background image to `site/public/images/hero-bg.jpg`

5. Run development server:
```bash
npm run dev
```

## Deployment

Deploy to Netlify at subdomain: `kelseys.mixedmakershop.com`

See `site/README.md` for detailed deployment instructions.
